## How to use this document
1) Pick a **base strategy** (one base in Funchal vs split stays).
2) For each day, pick a **region** based on weather (peaks vs coast).
3) Book your PR hikes in the official reservation system (see “Booking workflow” section).
4) Use the map pins to build 2–4 stop “loops” per day.
